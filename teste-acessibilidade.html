Teste de Funcionalidades de Acessibilidade

Instruções de Teste:

1. Teste os botões de fonte: Clique em A-, A, A+ e observe a mudança no tamanho do texto.

2. Teste o alto contraste: Clique no botão "Alto Contraste" e veja as cores mudarem.

3. Teste os atalhos de teclado:

  • Pressione Alt + Shift + C para alternar o contraste
  • Pressione Alt + Shift + F para ciclar entre os tamanhos de fonte

4. Teste a persistência: Altere as configurações, recarregue a página (F5) e veja se as preferências foram mantidas.

✅ Se você conseguir fazer tudo acima, a acessibilidade está funcionando!
